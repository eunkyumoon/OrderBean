# 문서

이 디렉토리는 프로젝트 관련 문서를 포함합니다.

## 파일 목록

- `PRD.md` - 제품 요구사항 정의서 (Product Requirements Document)
- `INSTALLATION.md` - 설치 및 실행 가이드
- `API.md` - API 엔드포인트 문서
- `DATABASE.md` - 데이터베이스 스키마 문서
- `DEPLOYMENT.md` - 배포 가이드
- `DEVELOPMENT.md` - 개발 가이드
- `TESTING.md` - 테스트 계획서 및 가이드
- `TEST_COVERAGE.md` - 테스트 커버리지 리포트
- `TEST_REPORT.md` - 테스트 실행 리포트
- `COVERAGE_REPORT.md` - 커버리지 상세 리포트
- `PROJECT_STATUS.md` - 프로젝트 현황 리포트
- `REPORTS_INDEX.md` - 리포트 인덱스
- `README.md` - 이 파일
