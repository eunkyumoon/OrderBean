## 📋 프로젝트 개요

**OrderBean**은 바쁜 직장인들을 위한 간편 모바일 선주문 웹 서비스입니다.

### 목표
- 카페 대기 시간 낭비 해소
- 복잡한 커스터마이징 주문의 어려움 해결
- 주문 히스토리 관리의 편의성 제공
- 주문 시간 3분 이상 단축
