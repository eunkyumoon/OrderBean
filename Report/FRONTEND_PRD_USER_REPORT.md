### 4. 페이지별 상세 요구사항 (와이어프레임 기반)

#### 3.1 랜딩 페이지 (`/`)
- 서비스 소개
- 주요 기능 요약
- CTA 버튼

#### 3.2 로그인/회원가입 (`/auth/login`, `/auth/register`)
- 인증 폼
- 유효성 검사
- 에러 처리

#### 4.1 메뉴 목록 (메인 화면) - 와이어프레임 기반
- **위치**: 메인 화면 (`/` 또는 `/menus`)
- **레이아웃**: 
  - 상단: 헤더 (COZY 로고 + 주문하기/관리자 버튼)
  - 중간: 메뉴 카드 그리드 (3열)
  - 하단: 장바구니 (고정 영역)
- **메뉴 카드 구조**:
  - 이미지 플레이스홀더
  - 메뉴명, 가격, 설명
  - 옵션 체크박스 (샷 추가 +500원, 시럽 추가 +0원)
  - "담기" 버튼
- **장바구니**:
  - 하단 고정 영역
  - 주문 항목 목록 (메뉴명 + 옵션 + 수량, 가격)
  - 총 금액 표시
  - "주문하기" 버튼

#### 4.2 장바구니 (하단 고정) - 와이어프레임 기반
- **위치**: 메인 화면 하단 고정
- **기능**:
  - 실시간 업데이트 (담기 클릭 시 즉시 반영)
  - 항목별 가격 표시
  - 총 금액 자동 계산
  - 주문하기 버튼

#### 3.6 주문 생성 (`/orders/new`)
- 주문 확인
- 결제 수단 선택
- 결제 처리

#### 3.7 주문 내역 (`/orders`, `/orders/:id`)
- 주문 목록 조회
- 상태 필터링
- 주문 상세 정보
- 재주문 기능

#### 3.8 레시피 (`/recipes`, `/recipes/new`, `/recipes/:id`)
- 레시피 목록
- 레시피 생성
- 레시피 상세
- 재주문 기능

#### 3.9 프로필 (`/profile`)
- 사용자 정보 표시
- 정보 수정
- 비밀번호 변경

| 구성 요소 | 설명 | 우선순위 |
|----------|------|---------|

| **메뉴 카드 그리드** | 3열 그리드, 옵션 선택 가능 | 높음 |

| 요소 | 설명 | 스타일 |
|------|------|--------|
| 이미지 | 플레이스홀더 또는 실제 이미지 | 150-200px 높이 |
| 메뉴명 | 굵은 글씨, 어두운 회색 | 18px, Bold |
| 가격 | 일반 글씨 | 16px, Regular |
| 설명 | "간단한 설명..." | 14px, Regular |
| 옵션 | 체크박스 (샷 +500원, 시럽 +0원) | 14px |
| 담기 버튼 | 연한 회색 배경, 회색 테두리 | 14px, Medium |

### 장바구니 구조

| 요소 | 설명 | 스타일 |
|------|------|--------|
| 제목 | "장바구니" | 일반 또는 굵은 글씨 |
| 항목 표시 | 메뉴명 + 옵션 + 수량, 가격 | 14px, Flexbox |
| 총 금액 | 굵은 글씨로 강조 | 16px, Bold |
| 주문하기 | 연한 회색 배경, 회색 테두리 | 16px, Medium |
